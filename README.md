 Library Management System (Java Swing + MySQL):

A simple and modern Library Management System built using Java Swing, JDBC, and MySQL.
It allows users to Add, View, Update, and Delete books using a clean graphical interface with a dark theme.
Features
*Add New Books:
Enter book details (ID, title, author, price) using a popup form.

*View All Books:
Displays all books from the database in a styled JTable.

*Update Existing Books:
Update any book by entering its book ID.

*Delete Books:
Remove books from the system easily.

*Modern Dark-Themed GUI:
Custom colors, hover effects, and a polished UI using Java Swing.

*MySQL Database Integration:
Fully integrated CRUD system using JDBC.
